**2. Función que calcula el factorial de un número:**

```python
def factorial(n):
  """Calcula el factorial de un número entero no negativo."""
  if n == 0:
    return 1
  elif n < 0:
    return "Factorial no definido para números negativos"
  else:
    factorial = 1
    for i in range(1, n + 1):
      factorial *= i
    return factorial

resultado = factorial(5)
print(f"El factorial de 5 es: {resultado}") # Output: El factorial de 5 es: 120
resultado = factorial(-2)
print(f"El factorial de -2 es: {resultado}") # Output: El factorial de -2 es: Factorial no definido para números negativos
```

**3. Función que verifica si una cadena es un palíndromo:**

```python
def es_palindromo(texto):
  """Verifica si una cadena de texto es un palíndromo (se lee igual al revés)."""
  texto = texto.lower() # Ignorar mayúsculas/minúsculas
  texto = ''.join(c for c in texto if c.isalnum()) # Eliminar espacios y caracteres especiales
  return texto == texto[::-1]

print(es_palindromo("Anita lava la tina")) # Output: True
print(es_palindromo("Python")) # Output: False
```

**4. Función que encuentra el elemento máximo en una lista:**

```python
def maximo_lista(lista):
  """Encuentra el elemento máximo en una lista."""
  if not lista:
    return None # Manejar lista vacía
  return max(lista)

lista_numeros = [1, 5, 2, 8, 3]
maximo = maximo_lista(lista_numeros)
print(f"El máximo de la lista es: {maximo}") # Output: El máximo de la lista es: 8

maximo = maximo_lista([])
print(f"El máximo de la lista vacía es: {maximo}") # Output: El máximo de la lista vacía es: None
```
